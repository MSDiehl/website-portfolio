AnimeHub (V1)

A centralized hub for discovering anime, tracking what you're watching, viewing detailed anime pages, and browsing an upcoming monthly release schedule all with a clean, modern UI.

React TypeScript .NET C# PortgreSQL

Highlights

  • Per-user authentication (create account, sign in/out) with user-specific tracking data
  • Search anime by title via AniList and track/untrack shows (persists in the database)
  • Detailed anime page with banner art, metadata, overview, and placeholder trend graphs (V1)
  • Weekly release schedule with date navigation + “Tracked only” filter for upcoming episodes
  • Anime discussions: create/read threads with optional episode scoping
  • Global forums route to “Coming soon” in V1 (planned Reddit-style feed)
